AppUser class

@Entity @Table(name = "USERS")
public class AppUser

JPA entity representing a registered application user.

Each row corresponds to a GitHub user who has authenticated at least once. The GitHub login is used as the natural primary key because it is unique across GitHub and is already used as the principal name throughout the security layer.

Instances are created automatically by UserService#findOrRegister on first login; they are never created manually.

See also: UserRole, UserService
