UserRoleRepository interface

public interface UserRoleRepository extends JpaRepository<UserRole, Long>

Spring Data JPA repository for UserRole entities.

Provides standard CRUD operations and a custom query to fetch all roles belonging to a specific GitHub login.

This repository is called only from UserService; no other class should inject it directly.
