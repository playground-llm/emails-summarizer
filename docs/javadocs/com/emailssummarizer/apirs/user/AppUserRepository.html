public interface AppUserRepository extends JpaRepository<AppUser, String>

Spring Data JPA repository for AppUser entities.

Provides standard CRUD operations inherited from JpaRepository. The primary key type is String (the GitHub login).

This repository is called only from UserService; no other class should inject it directly.

See also: UserService

Method Summary
