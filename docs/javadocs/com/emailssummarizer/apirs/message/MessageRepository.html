MessageRepository interface

@Repository
public interface MessageRepository extends JpaRepository<Message, UUID>

Spring Data JPA repository for Message entities.

Provides standard CRUD operations inherited from JpaRepository plus custom query methods that operate on the categoryCode foreign key. This repository must only be called from MessageService and CategoryService — controllers must not interact with it directly.

See also: MessageService
