MessageRequest record

public record MessageRequest( String title, String body, String categoryCode )

Immutable DTO carrying the data required to create or update a Message.

Used as the request body for POST /messages and PUT /messages/{id}. On update, the categoryCode component is accepted but currently ignored by the service layer — only title and body are modified.

Validation of individual fields (non-blank title, existence of the referenced categoryCode) is performed by MessageService, not at the DTO level.

See also: MessageService

Record Components

title String
The subject line of the message; must not be null.
body String
The full content of the message; may be null.
categoryCode String
The code of the category this message belongs to.
Must reference an existing Category on creation. Ignored on update — the category assignment of an existing message cannot be changed.
