MessageController class

@RestController @RequestMapping("/messages")
public class MessageController

REST controller that exposes the message management API under /messages.

Handles HTTP request/response mapping only; all business logic is delegated to MessageService. This class must not interact with any repository directly.

Requires the following roles depending on the HTTP method:

  • ROLE_READ — GET /messages
  • ROLE_EDIT — POST /messages, PUT /messages/{id}
  • ROLE_DEL — DELETE /messages/{id}
See also: MessageService
