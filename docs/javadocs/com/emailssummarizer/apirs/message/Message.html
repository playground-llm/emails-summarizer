Message class

public class Message

JPA entity representing an individual email message.

Maps to the MESSAGE database table. Each message belongs to exactly one Category, referenced by the categoryCode foreign key. The primary key is a UUID generated by the database at insert time.

The body field is stored as a CLOB to support large message content.

See also: MessageService, MessageRepository
