CategoryController class

@RestController @RequestMapping("/categories")
public class CategoryController

REST controller that exposes the category management API under /categories.

Handles HTTP request/response mapping only; all business logic is delegated to CategoryService. This class must not interact with any repository directly.

Requires the following roles depending on the HTTP method:

  • ROLE_READ — GET /categories
  • ROLE_EDIT — POST /categories, PUT /categories/{code}
  • ROLE_DEL — DELETE /categories/{code}
See also: CategoryService
