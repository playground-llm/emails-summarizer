Category class

@Entity @Table(name = "CATEGORY")
public class Category

JPA entity representing a message category.

Maps to the CATEGORY database table. The code field is a unique business key used as the URL path parameter in REST endpoints and as a foreign key target from Message. Once set, code must not be changed — it is treated as immutable after creation.

The id is an auto-generated surrogate key used internally by JPA; external callers should always refer to categories by their code.

See also: CategoryService, CategoryRepository
